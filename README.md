# Fallen Ascendants

Auto Battle TCG (Turn-Based Strategy Card Game) developed using Java and LibGDX as a final project for Data Structures.

## Overview

Fallen Ascendants is a card-based auto battle game set in a dark fantasy world where celestial beings have fallen from grace and various factions compete for divine power.

Players collect cards, build decks, arrange formations, and battle against AI opponents in an automated turn-based combat system.

## Core Features

- Deck Building
- Auto Battle System
- Turn-Based Combat
- SPD (Speed) Queue System
- Aggro Targeting System
- Card Collection
- Card Upgrade System
- Reward Selection System
- Save Progress
